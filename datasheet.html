Project Title

Measuring Fairness

Author

Google PAIR (People + AI Research)

Date Published

2018

Project Type

Interactive Explorable Explanation

Topics

  • Algorithmic Fairness
  • Human-Centered Machine Learning
  • Data Ethics

Description

“Measuring Fairness” is a pedagogical web project that explains various formal definitions of fairness in machine learning. It demonstrates through interactive simulations how these definitions can conflict with one another and what trade-offs must be made in applying them.

Technologies Used

  • JavaScript
  • D3.js
  • HTML/CSS

Project Goal

To educate ML practitioners, students, and the broader public about the nuances of fairness in algorithmic systems, especially how “fairness” is not a single concept but a set of mathematically incompatible metrics.

Intended Audience

  • Data Scientists
  • Policy Makers
  • Researchers
  • General Public
