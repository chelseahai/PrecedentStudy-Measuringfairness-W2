What Exists in the Project

The "Measuring Fairness" project consists of multiple conceptual and interactive elements, including:

  • Fairness Metrics: Demographic parity, equalized odds, predictive parity
  • Trade-offs: Highlighting incompatibilities between metrics
  • Data Abstractions: Simulated datasets to emphasize concept over context
  • Interactive Sliders: Threshold controls that simulate bias adjustment
  • Visual Outputs: Confusion matrices, ROC curves, distribution plots
  • Narrative Modules: Step-by-step guidance through fairness logic

Component Breakdown

Each module within the explorable serves a teaching function:

  • Intro Module: Defines fairness and sets user expectations
  • Interactive Panels: Let users manipulate variables and see consequences
  • Explanatory Narration: Text panels contextualize what users are seeing
